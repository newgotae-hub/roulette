이벤트 추첨 전 체크리스트

추첨 도구를 준비해도 시작 직전에 놓치는 항목이 많습니다. 특히 행사 진행에서는 명단 누락, 중복 입력, 당첨 제외 규칙 미공지, 결과 저장 누락이 반복됩니다. 아래 체크리스트는 추첨을 누르기 전에 반드시 확인해야 하는 최소 항목입니다.

1. 명단 검수

  • 참가자가 모두 포함되었는가?
  • 이름 철자나 번호가 최신 정보와 일치하는가?
  • 같은 참가자가 두 번 이상 들어가 있지 않은가?
  • 팀 단위 추첨인지 개인 단위 추첨인지 목록 형태가 맞는가?

명단 검수는 가장 먼저 끝내야 합니다. 추첨 후에 명단 문제가 발견되면 결과보다 운영 실수가 더 크게 남습니다.

2. 규칙 공지

  • 당첨자를 다음 라운드에서 제외할 것인가?
  • 중복 당첨이 가능한가?
  • 추첨을 다시 돌릴 수 있는 조건은 무엇인가?
  • 한 번에 한 명씩 뽑는지, 여러 명을 연속 추첨하는지?

규칙은 진행자가 알고 있는 것만으로는 부족합니다. 참가자도 같은 내용을 알아야 합니다. 시작 전에 규칙을 짧게 읽고, 화면에도 그대로 보여주는 편이 안전합니다.

3. 화면 상태

모바일이나 방송 화면에서는 UI 일부가 가려지기 쉽습니다. 추첨 전에는 명단, 설정, 추첨 버튼, 결과 기록 영역이 모두 보이는지 확인하세요. 전체화면 기능을 쓸 경우에는 사전에 한 번 켜 보고, 다시 돌아오는 동작도 확인하는 편이 좋습니다.

4. 기록 저장 준비

추첨이 끝난 뒤 파일을 저장하려고 하면 정신이 없어서 놓치기 쉽습니다. 미리 결과를 어떤 방식으로 남길지 정해 두세요. CSV로 저장할 것인지, 방송 화면을 캡처할 것인지, 진행자가 별도 로그를 남길 것인지가 정리되어 있어야 이후 문의 대응이 쉬워집니다.

5. 테스트 1회

실제 행사 전에 테스트를 한 번 돌려보는 것이 가장 비용이 적은 예방책입니다. 사운드가 너무 크지 않은지, 명단이 제대로 반영되는지, 당첨자 제외가 예상대로 동작하는지, 결과 기록이 쌓이는지까지 한 번 확인하면 실전 오류를 크게 줄일 수 있습니다.
