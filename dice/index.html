이 도구는 무엇인가요?

주사위 굴리기는 브라우저에서 바로 실행되는 랜덤 추첨 도구입니다. 별도 설치 없이 이벤트, 수업, 팀 활동에서 빠르게 사용할 수 있습니다.

입력 데이터와 결과 상태는 화면에서 즉시 반영되며, 모바일과 데스크톱에서 모두 읽기 쉬운 레이아웃으로 제공됩니다.

FAQ

모바일에서도 동일하게 동작하나요?

네. 모바일 화면에 맞춰 레이아웃이 자동으로 정리됩니다.

기록을 저장할 수 있나요?

네. 추첨 기록은 CSV로 내려받아 관리할 수 있습니다.

입력 형식 제한이 있나요?

줄바꿈, 공백, 쉼표 등 일반적인 입력 구분을 지원합니다.

입력 데이터는 외부로 전송되나요?

아니요. 데이터는 브라우저 내부에서 처리됩니다.
