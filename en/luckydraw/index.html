FAQ

Can I use custom names or numbers?

Yes. Select Custom List and enter items separated by spaces, commas, tabs, or line breaks.

Can I sort the results?

Yes. Use Ascending to sort results numerically or alphabetically.

How do I save the history?

Click Export CSV to download all draw sets as a spreadsheet file.

Is my data safe?

Yes. All processing is local in your browser. No input is sent to external servers.

How to Use

  • Prepare the input list or values you want to use in Lucky Draw.
  • Adjust draw options and run settings for your use case.
  • Run the draw, review the result, and continue with additional rounds if needed.
  • Check the history panel and export results as CSV.

What Is This Tool?

Lucky Draw is a browser-based random selection tool for events, classes, team activities, and quick decisions.

Your input and result state are reflected instantly in the page and can be reviewed comfortably on both mobile and desktop.

FAQ

Does it work well on mobile?

Yes. The layout adapts to small screens automatically.

Can I save the result history?

Yes. You can export result history as CSV.

Are common input separators supported?

Yes. Common separators like line breaks, spaces, and commas are supported.

Is my data sent to external servers?

No. Data is processed locally in your browser.
